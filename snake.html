Snake

By Steven Chang

I recreated this classic game for an assignment based on JavaScript. I'm pretty happy with the game play and the look of the game. Had to get a bit crafty to get the look of the snake right and also to increase the speed of the game as the snake eats more mice.

I used Balsamiq for the mockups, bootstrap as the html framework, sass to help organise the css and jQuery/JavaScript for the actual game.

I can pretty much nail the top score every try now. If I was to progress with this, I'd give people options for different sized boards. It would be nice to set up the backend so that the top-scores could be permanently kept so that other people can try to beat them. Also, I would make a different scoring system as well where time would also be a factor, as in the most number of mice in the shortest amount of time/moves.
